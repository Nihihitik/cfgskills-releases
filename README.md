# cfgskills

  **cfgskills** — десктопное приложение для управления «скиллами» (директориями со `SKILL.md`) для Codex и Claude Code. Позволяет просматривать, искать, копировать и удалять скиллы между двумя локальными каталогами через единый интерфейс.

  > Репозиторий релизов.

  ## Загрузка

  Скачать последнюю версию можно со страницы **[Releases](https://github.com/nihihitik/cfgskills-releases/releases/latest)**.
  | Платформа | Файл |
  |---|---|
  | macOS (Apple Silicon) | `cfgskills_<version>_aarch64.dmg` |
  | macOS (portable) | `cfgskills.app.tar.gz` |
  | Windows 10/11 (x64) | `cfgskills_<version>_x64-setup.exe` |


  ### Установка

  **macOS:** откройте `.dmg`, перетащите `cfgskills.app` в `Applications`.

  **Windows:** запустите `.exe`-инсталлятор и следуйте подсказкам мастера.

  ## Возможности

  ### Управление скиллами
  - **Единый каталог** скиллов из двух источников:
    - Codex — `~/.codex/skills`
    - Claude Code — `~/.claude/skills`
  - **Поиск** по имени и содержимому внутри каждого каталога.
  - **Предпросмотр `SKILL.md`** в отдельном диалоге — без необходимости открывать редактор.
  - **Копирование скиллов** между Codex и Claude Code в один клик с сохранением относительной структуры директорий.
  - **Удаление скиллов** с автоматической очисткой пустых родительских папок.
  - **Открытие в проводнике** (Finder на macOS, Explorer на Windows) для ручного редактирования.

  ### Интерфейс
  - **Светлая и тёмная темы** + автоматическое следование системной.
  - **Переключение языка** — русский и английский.
  - **Боковая навигация** между разделами Codex, Claude Code и настройками.

  ## Системные требования

  - **macOS:** 11 Big Sur или новее, Apple Silicon (M1/M2/M3/M4).
  - **Windows:** Windows 10 64-bit (версия 1809+) или Windows 11.
